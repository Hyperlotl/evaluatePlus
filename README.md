# evaluate+
Write and run JavaScript within Scratch! 

This extension is EXTREMELY EXPERIMENTAL.
## features:
1. a custom pseudo-IDE to write JS in
2. the ability to execute your written javascript functions using scratch blocks
3. your custom javascript functions CAN call hat blocks in the reporter
