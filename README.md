# evaluate+
Write and run JavaScript within Scratch! 

This extension is EXTREMELY EXPERIMENTAL.
## features:
1. Write custom JavaScript functions and run them in Scratch! These functions are more powerful than odinary Scratch functions
2. Custom Javascript functions can trigger events in the project, namely by using `triggerHats()`
3. Custom Javascript functions can get and set variables in the project, using `getVar()` and `setVar()` respectively.
4. A semi-fledged(?) pseudo-ide to code JS functions in!
5. Your JavaScript functions are stored in extension storage(i.e they are saved in the project file)
6. This extension is **fully compatible** (as far as I have tested anyways :3) with the TurboWarp packager.
